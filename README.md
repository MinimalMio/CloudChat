# CloudChat  

[简体中文](README_ZH.md)
A open-source text messaging application. (*^▽^*)   

## Build  

### Server  
You need a Linux environment with gcc and make.  
```Bash
./configure --enable-server && cd server && make && sudo make install
```

### Client
You need a Linux environment with gcc and make.  
```Bash
./configure --enable-client && cd client && make && sudo make install
```
